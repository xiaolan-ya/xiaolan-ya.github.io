充值 ​

余额用于支付模型请求费用，余额为 0 时调用会失败。第一次接入建议先充小额（¥10 即可），跑通后再按用量充值。

第 1 步：打开钱包页面 ​

登录控制台后，点左侧导航栏的 钱包。页面上方是当前余额，中间的 添加资金 区域就是充值入口：

第 2 步：选择充值金额 ​

在 金额 区域点一个预设档位（¥10 / ¥20 / ¥50 / ¥100 / ¥200 / ¥500，图中 ①），或在 自定义金额 框里填其他数值（图中 ②，右侧会实时显示「待支付金额」）。

第一次接入只需要先充 ¥10 把流程跑通，后续再按需充值。

第 3 步：选择支付方式并支付 ​

在 付款方式 选择 支付宝 或 微信（图中 ③），然后按页面提示完成支付（扫码或跳转支付）。

支付成功后余额即时到账，无需等待，回到钱包或概览页即可看到余额变化。

有兑换码？

付款方式下方有「兑换码」输入框，填入兑换码后点 兑换额度 即可直接到账，无需走支付。

充值后确认 ​

检查项在哪看
余额是否到账钱包页顶部 当前余额，或概览页的余额卡片
这笔充值记录钱包页右上角 订单历史 按钮

充了钱但余额没变？ 先刷新页面，再点 订单历史 看有没有这笔记录：

  • 订单历史里有记录且已支付 → 等几秒后刷新概览页；
  • 订单历史里没有这笔 → 说明支付没有真正完成，请重新发起。

下一步 ​

余额到账后：

  1. 创建专属 Key（如果还没创建）
  2. 去 模型广场 点模型名复制
  3. 配置工具开始使用，详见 快速接入
