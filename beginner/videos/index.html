视频参考 ​

视频用于确认界面位置和安装步骤。命令、下载地址和参数以文档为准。

看视频更有用的场景通常是：

  • 想确认按钮、菜单和安装界面长什么样
  • 想看清某个系统弹窗、权限提示或下载页面
  • 明明看懂了命令，但不知道它应该在哪个位置输入

真正在本机执行命令、填写地址和复制模型名时，仍然以正文文档和官方入口为准。

安全检查 ​

  • 不运行来源不明的远程脚本。
  • 不下载二次打包安装包。
  • 不把 API Key 发给第三方。
  • 不在录屏或截图里展示完整 Key。
  • 不使用要求关闭安全软件的未知工具。
  • 视频里的命令和版本可能过期，执行前以本文档和官方入口为准。
